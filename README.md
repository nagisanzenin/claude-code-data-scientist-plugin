# Data Scientist Plugin for Claude Code

AI/ML/LLM optimization -- prompt engineering, token optimization, A/B testing, and scientific studies.

## Installation

```
/plugin marketplace add nagisanzenin/claude-code-plugins
/plugin install data-scientist@nagisanzenin
```

## Features

- Prompt engineering and optimization
- Token usage optimization
- A/B testing frameworks
- ML model evaluation pipelines
- Scientific study design
